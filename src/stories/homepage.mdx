# Welcome to Aspyn UI Kit

This is the UI component library for Aspyn project. Here you will find all the reusable components that can be used across different projects.

## Getting Started

To get started with using the components, you can browse through the stories in the sidebar. Each story demonstrates a component with various states and configurations.

## Installation

To install the UI Kit in your project, run:

```bash
npm install @aspyn-io/uikit
```

## Usage

Here is an example of how to use a component from the UI Kit:

```tsx
import { Button } from '@aspyn-io/uikit';

const App = () => (
  <div>
    <Button label="Click Me" />
  </div>
);

export default App;
```
